# Bank Customer Churn Prediction

This project focuses on predicting bank customer churn using machine learning techniques. The main objective is to identify customers who are likely to leave the bank and support better customer retention strategies through data-driven insights.

## Project Overview
The project uses a structured bank customer dataset and applies multiple classification models to compare predictive performance. The workflow includes data preprocessing, feature engineering, model training, evaluation, and interpretation of results.

## Models Used
- Logistic Regression
- Random Forest
- XGBoost

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
- Seaborn

## Methodology
- Data cleaning and preprocessing
- Handling categorical variables
- Feature scaling
- Train-test split
- Model training and comparison
- Performance evaluation using Accuracy, Precision, Recall, F1-score, and ROC-AUC
- Confusion matrix analysis
- ROC and Precision-Recall curve plotting
- Threshold tuning
- Feature importance analysis

## Key Findings
- XGBoost delivered the best overall performance
- Random Forest also performed strongly for churn classification
- Logistic Regression provided a useful baseline model
- Important features included customer balance, tenure, and credit score
- Threshold tuning helped improve business-focused model interpretation
